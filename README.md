# Sistema de Gerenciamento de Veículos

Projeto final da disciplina de Programação Orientada a Objetos (POO) - CEFET-MG.
Aplicação em Java com interface gráfica (Swing) para registrar, buscar e remover veículos (carros e motos).

## Conceitos aplicados
- Orientação a Objetos
- Tratamento de Exceções
- Manipulação de Arquivos
- Interface Gráfica (Swing)
